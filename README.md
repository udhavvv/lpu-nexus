**Institution:** Lovely Professional University

## Overview
LPU Nexus is a comprehensive, AI-powered student lifecycle management platform designed for large-scale higher education institutions. Built using Python, Streamlit, and the Anthropic Claude Sonnet 4 API, it unifies nine functional modules to handle academic risk detection, skill development, peer networking, and mental health support.

## Key Modules
1. **AcadRadar:** Real-time semester survival probability and AI rescue plans.
2. **SkillDNA:** Multi-dimensional skill-graph synthesis and archetype classification.
3. **FirstSem AI:** Course-aptitude compatibility assessment.
4. **SkillForge:** Gamified, branch-calibrated challenges with XP mechanics.
5. **CampusConnect:** Interest-topology peer matching.
6. **ProjectHive:** Collaborative project lifecycle and team formation.
7. **AskTeacher:** Faculty-expertise routing and peer mentor matching.
8. **MindSpace:** Anonymous mental health triage with safe-messaging protocols.
9. **Nexus AI:** Campus-aware conversational assistant.

## Setup Instructions
1. Clone this repository.
2. Create a `.env` file based on `.env.example` and add your API keys.
3. Install dependencies: `pip install -r requirements.txt`
4. Run the application: `streamlit run app.py`
